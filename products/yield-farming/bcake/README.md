# bCAKE

<figure><img src="../../../.gitbook/assets/bCAKE page.webp" alt=""><figcaption></figcaption></figure>

bCAKE - 农场收益助推器，具备自动计算、农场收益加成双重功能，可提升指定农场的 CAKE 产量，最高至 2.5 倍。这是一种充分利用 veCAKE 的优势并与农场流动性凭证质押结合的产品。

**bCAKE 不是代币，不能转让！**
